# Finding Numbers in a String (C# Console Application)

## Description
This console application was my first assignment as a .NET development student.  
The task was to create an algorithm that identifies numbers within a string of text.

The application reads user input, extracts numeric substrings based on defined rules, highlights valid numbers in the console output, and calculates their total sum.

## Tech Stack
- C#
- .NET (Console Application)

## Features
- Parses a text string to identify numeric values
- Filters numbers based on specific criteria
- Highlights valid numbers in the console using colors
- Calculates and displays the total sum of all valid numbers
